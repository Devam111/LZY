# LZY – Learnsy Online Learning Platform

LZY is a full-stack online learning platform where students and instructors can register, create courses, enroll, and track learning progress.

---

## Project Structure

LZY/
├── Learnsy (Frontend)
└── Learnsy-backend (Backend)

---

## Tech Stack

Frontend:
- React + Vite
- Tailwind CSS

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
